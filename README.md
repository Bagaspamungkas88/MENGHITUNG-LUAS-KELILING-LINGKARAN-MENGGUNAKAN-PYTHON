# MENGHITUNG-LUAS-KELILING-LINGKARAN-MENGGUNAKAN PYHTON

1. mulai
2. input r 
3. proses masukan phi 22/7
4. hituang luas p=phi*r*r hitung keliling=2*phi*r
5. tampilkan luas dankeliling lingkaran 
6. selesai
